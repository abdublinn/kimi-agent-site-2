# Kimi Agent - Site by Requirements

Landing page built with React, TypeScript, Vite, Tailwind CSS, and shadcn/ui.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deployment

The repository includes GitHub Actions workflow at `.github/workflows/deploy-pages.yml`.
On every push to `main`, it builds the app and deploys `dist` to GitHub Pages.
